\section{Estabilidad a largo plazo}
\label{sec:estabilidad}

En sistemas de procesado de vídeo en continuo es habitual que, con el tiempo,
la CPU vaya consumiendo cada vez más sin motivo aparente o que la memoria
crezca de forma silenciosa hasta agotar los recursos del equipo. Estos
problemas, conocidos como \textit{CPU creep} y fuga de memoria~\cite{huang1995software}, no aparecen
en pruebas cortas y solo se detectan dejando el sistema en marcha durante un
periodo prolongado. El objetivo de esta prueba es comprobar que Voctomix no
presenta ninguno de estos comportamientos.

Se arrancó el sistema completo en Docker~Compose con las cuatro cámaras activas
y se dejó funcionando 31 minutos seguidos. Las fuentes de vídeo se generaron
con FFmpeg directamente en el equipo para que nada externo pudiera fallar.
Cada 5 segundos la telemetría anotaba el uso de CPU y RAM, acumulando 377
muestras en total.

de arranque y se mantuvo estable sin ninguna tendencia creciente. La RAM se mantuvo

ningún fallo, lo que confirma que el sistema es apto para retransmisiones de
larga duración sin necesidad de supervisión continua.

\section{Resiliencia y recuperación}
\label{sec:resiliencia}

Se midió cuánto tarda el sistema en recuperarse de forma autónoma cuando una cámara falla.
Para ello se mató el proceso principal del contenedor cam1 con \texttt{kill -9}
simulando un fallo real, y se cronometró el tiempo hasta que Docker lo reinició
y lo declaró \textit{healthy}. Se repitió 10 veces.

    \end{axis}
  \end{tikzpicture}

           (el eje Y empieza en 510\,ms para mostrar la variación entre mediciones).}
  \label{fig:resiliencia}

El sistema se recuperó siempre en torno a 520\,ms, con una variación de
apenas 10\,ms de principio a fin. Este resultado es positivo para una retransmisión en directo: si una cámara
cae en medio de una emisión, el sistema la devuelve automáticamente en medio
segundo, sin que nadie tenga que intervenir. El espectador apenas lo notaría, ya que el mezclador
puede seguir emitiendo con las otras cámaras mientras tanto.
